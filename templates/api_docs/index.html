Authentication

Most API endpoints require authentication. The API supports the following authentication methods:

  • JWT Bearer Token: Include the token in the Authorization header as Bearer <token>
  • Cookie Authentication: For browser-based applications, authentication cookies are set automatically when logging in

Obtaining Authentication Tokens

To obtain an authentication token, use the /auth/login endpoint with your credentials.
